My Courses

Course Name Course Code Course Description
Problem Solving and Programming CSC 1100 Focuses on problem solving with algorithms, and their realization as computer programs. Topics include data types, operators, expressions, assignment, input and output, control structures, modularity, and procedural abstraction using functions.
Computer Science I CSC 2110 Introduces fundamental object-oriented concepts and programming techniques. Covers data abstraction, recursion, programming with generic data types, inheritance, polymorphism, and exception handling. Emphasizes console programs and event-driven programming using a simple graphics API.
Fundamental Structures in Computer Science CSC 1500 Introduces fundamental control and data structures such as algorithms, recursive algorithms, program correctness, mathematical induction, and divide-and-conquer algorithms. Topics also include computing with relations and graph/tree properties.
Computer Science II CSC 2200 Explores design and implementation of abstract data types, including stacks, queues, trees, lists, hashing, and graphs. Topics also cover recursion, sorting, searching, hashing, string processing, and algorithm analysis.
Calculus I MAT 2010 Studies calculus as the study of change, covering differentiation, integration, and their applications. Includes concepts of the derivative, the definite and indefinite integrals, and their real-world interpretations.
Reliability and Engineering Statistics ET 3850 Introduces probability distributions (binomial, Poisson, normal), confidence intervals, linear regression, and statistical quality control. Focuses on the application of statistics in engineering reliability.
Microprocessor Fundamentals EET 2720 Covers the use of microprocessors, including software, interfaces, memory, registers, and system architecture. Provides hands-on programming design projects focused on microprocessor-based systems.
